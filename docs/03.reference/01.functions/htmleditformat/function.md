Replaces special characters in a string with their
        HTML-escaped equivalents.
        [version]
            HTML version to use. currently ignored.
            -1: The latest implementation of HTML
            2.0: HTML 2.0 (Default)
            3.2: HTML 3.2
            4.0: HTML 4.0
