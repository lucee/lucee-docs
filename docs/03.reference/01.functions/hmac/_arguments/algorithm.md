The algorithm to use for hashing your input, default is `HmacMD5`.

The available algorithms (depending on your JVM) are

- HmacSHA512
- HmacSHA3-384
- HmacSHA384
- HmacSHA3-512
- HmacSHA512/256
- HmacSHA3-224
- HmacSHA512/224
- HmacSHA3-256
- HmacSHA256
- SslMacMD5
- HmacSHA224
- SslMacSHA1
- HmacSHA1
- HmacMD5
