### Non-Member Function

```luceescript+trycf
animals = {
	cow: "moo",
	pig: "oink",
	cat: "meow"
};

Dump(
	label: "Current animals",
	var: animals
);

// Update cat in animals
StructUpdate(animals, "cat", "purr");

Dump(
	label: "Animals with cat updated",
```
