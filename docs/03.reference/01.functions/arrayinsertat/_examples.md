### Insert item into an array at a given position within the array length

```luceescript+trycf
fruitArray = ['apple', 'kiwi', 'banana', 'orange', 'mango', 'kiwi'];

arrayInsertAt(fruitArray,3,'new inserted item');
dump(fruitArray);

// member function
fruitArray.insertAt(3,'member added item');
dump(fruitArray);
```
