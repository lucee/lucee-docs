This function is deprecated, use instead [[function-IsEmpty]].

This function returns `true` if a value exists and is not "empty".

The following things are considered to be empty:

* `""` (an empty string)
* `[]` (an empty array)
* `{}` (an empty struct)

Any other value returns `false`
